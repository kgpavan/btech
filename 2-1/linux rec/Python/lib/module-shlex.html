5.11 shlex -- Simple lexical analysis

New in version 1.5.2.

The shlex class makes it easy to write lexical analyzers for simple syntaxes resembling that of the Unix shell. This will often be useful for writing minilanguages, e.g. in run control files for Python applications.

shlex ([stream])
A shlex instance or subclass instance is a lexical analyzer object. The initialization argument, if present, specifies where to read characters from. It must be a file- or stream-like object with read() and readline() methods. If no argument is given, input will be taken from sys.stdin.
